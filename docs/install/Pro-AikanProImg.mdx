# AikanProImg

## 前言准备

 :::caution 安装环境

- 准备一个（子）域名，新建站点，不需要创建数据库和ftp
- **根目录设置为：/opt/files/static**
- **然后根据自己的需要配置SSL<font color="red"> 这点很重要，关系这以后社区图片的访问问题</font >**
:::
