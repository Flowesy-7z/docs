## 全新部署

:::tip 安装环境

由于软件运行环境原因，系统必须安装以下环境，请提前部署，以免在正式安装造成安装问题.

如果你不知道你的环境到底是 `64 位` 还是 `32 位`，一般来说就是 `64 位`。

需要用到的环境：
- PHP： 7.2（需要安装插件SG15和Redis）[SG15 如何安装](/question/sg15/)
- Redis：（5.0-7.0）推荐6.0，Redis是现在最受欢迎的NoSQL数据库之一。
- Mysql：（5.5-5.7）推荐5.6
- Nginx：稳定版即可，没有版本要求
:::
