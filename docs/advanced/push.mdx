# 消息推送配置
## 配置unipush消息通知
#### 目前AikanPro使用的是unipush1.0版本，所以在打包时需要勾选Push模块1.0。 然后前往dcloud开发者中心：[点击进入](https://dev.dcloud.net.cn/pages/app/list)，点击应用后根据下图所属开通，证书签名如果是云端证书就去云端证书那里拿，如果是自己的证书就百度怎么获取

<img  src="https://tk.illlt.com:759/aikanpro/docs//img/advanced/advanced-push-01.png"   />
设置好之后，根据下图获取圈出来的三个，填写到Star后台的推送设置就好了，顺序是124,3不需要
<img  src="https://tk.illlt.com:759/aikanpro/docs//img/advanced/advanced-push-02.png"   />
