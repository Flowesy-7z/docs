# API 命令表

##AIkanPro 拥有两套命令表，大家可以根据自己实际情况来用，第二种比较方便简单

#### 第一种是大佬命令：[点我跳转第一种命令](#第一种大佬命令)
#### 第二种是新手命令（推荐）：[点我跳转第二种命令](#第二种新手命令)

## 第一种大佬命令
### 命令表分很多种情况，一般用的最多的是重启
:::tip API 启动命令  
```bash showLineNumbers
sh /opt/aikanproapi.sh start
```
:::
:::tip API 重启命令  
```bash showLineNumbers
sh /opt/aikanproapi.sh restart
```
:::

:::tip API 停止命令  
```bash showLineNumbers
sh /opt/starapi.sh stop
```
:::

:::tip API 查看帮助信息命令  
```bash showLineNumbers
sh /opt/aikanproapi.sh help
```
:::

## 第二种新手命令
### 此命令比较简单，首先打开命令行任意路径输入以下命令回车后即可成功
:::tip API 查看帮助信息命令  
```bash showLineNumbers
wget -O ak https://aikan.pro/api/aikanapi.sh && mv aikanapi.sh /usr/local/bin/ak && chmod +x /usr/local/bin/ak && ak
```
- **除了回车之外，什么都不用操作，很简单的**
- **下次还想使用新手模式直接在命令行  <font color="red">任意路径</font>  输入 <font color="red">ak</font> 即可**
