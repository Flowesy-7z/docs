# **图标转跳配置教程**

## 常用链接
:::tip
- 商城首页：/pages/shop/shop（可不登陆）
- 邀请页面：/pages/user/rebate（需登录）
- 签到页面：/pages/user/signin（需登录）
- 任务页面：/pages/user/task（需登录）
- 蓝V申请：/pages/user/identifyblue（需登录）
- 实名认证：/pages/user/identify（需登录）

:::

## 自定义链接
### 其他的独立页面都支持转跳，至于想知道某页面的链接，可以打包H5，然后通过访问H5来获取（#号后面的地址）即是转跳地址
#### 比如商城链接：

<img  src="https://tk.illlt.com:759/aikanpro/docs//img/advanced/advanced-icon-01.jpg"   />

如果经验比较丰富，可以直接去APP源码的pages目录里面找自己想要的页面。
 :::caution 注意事项
- 部分页面是需要用户登录后才能正常显示，所以添加需要登录页面时候要做好转跳限制，如果你不知道需不需要登录，可以自行测试会不会出现数据错误，或直接无脑强制登录，限制登录没有坏处。
- 图标转跳目前仅支持APP内页面转跳，并且不支持转跳包含底栏的4个页面，比如首页，圈子页，消息页，我的页。
- 支持页面带参数转跳，比如转跳到官方公告的圈子，并且知道这个圈子id为1，转跳链接就可以设置为：/pages/forum/home?id=1

 :::
