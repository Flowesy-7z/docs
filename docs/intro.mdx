## 软件特色

* 美化了大量页面UI排版，APP整体风格简约、唯美。
* 拥有独立的管理后台，可远程配置APP的UI排版、模块开关、功能，管理整站数据等... 
* 完整的圈子模块、圈主权限体系，适合用来运营交友社区，校园论坛等...
* 用户个性化系统：用户背景、性别、实名、蓝V认证等功能...
* 更完善的推送：私信消息，系统通知，帖子消息，商品定购推送...
* 拥有完整的任务系统、签到系统模块，促使用户活跃。
* 同时挂载多个存储策略
* 支持  阿里云 OSS, FTP,本地存储,  腾讯云 COS
* 更多功能待你探索发现...

## 技术结构

### 爱看Pro 系统Java + PHP，基于Java+Vue+php+Uni-app开发

### 什么是uniapp？

*uni*-*app*（*uni*，读yo*u ni*，是统一的意思）是一个使用Vue.js开发所有前端应用的框架，开发者编写一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序、快应用等多个平台。

官网地址：[点击进入](https://uniapp.dcloud.io/)

移动端 uniapp 开发 使用 HbuilderX 开发

------

### 运行环境及框架：

1. 移动端uniapp开发框架 可生成APP H5 公众号 微信小程序
2. WEB Pc 管理后台使用Vue + Element UI 开发 兼容主流浏览器 ie11+
3. 后台接口 Java SpringBoot + Mybatis-plus + Mysql + redis
4. 后台管理 php 7.2 SG15加密
5. 运行环境 linux和windows等都支持,只要有Java环境和对应的数据库 redis
6. 运行条件 Java 1.8 Mysql5.7
