#  SG15 加密组件安装

###  宝塔面板 PHP7.2（我们目前使用的版本） 组件 SG15加密 暂末更新 请手动安装

###  这里教搭建两个方法来安装

### 第一种：

:::info sg15安装方法

1. 就是纯手动安装，找到宝塔指定目录进行替换即可！（以宝塔面板最新版本为例）

2. /www/server/php/72/lib/php/extensions/你的版本目录/ixed.7.2.lin 正常情况里面就一个文件夹放入后 重启PHP即可

3. 在 PHP-7.2管理 -> 配置文件 -> 最后一行加入下面这行配置即可

4. extension=/www/server/php/72/lib/php/extensions/no-debug-non-zts-20170718/ixed.7.2.lin

   :::
