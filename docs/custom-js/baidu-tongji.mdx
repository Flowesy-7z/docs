# 百度统计代码如何添加到 ZFile

登录 [百度统计](https://tongji.baidu.com/) 后点击上方的 "使用设置"，在网站列表右侧点击 "获取代码"，然后复制代码添加到 ZFile 的 "自定义 JS" 中即可。
